# pixelate-redactor

Redact images with blur

Works with Python >= 2.6, Python >= 3.2.

## Installation

    pip install pixelate-redactor

## Example

    pixelate-redactor --input=img/bps.jpg --output=img/bps.png --pixel-size=10 --start-x=100 --start-y=100 --size-x=500 --size-y=60
